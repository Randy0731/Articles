## 一句話結論

大叔把 **ONDS** 2025-10 後的賣壓重新歸因於 2022-2024 舊可轉債的 reset / acceleration overhang，並用官方 2025-07-18 all convertible notes retired 與 2026-01 `$1B` above-market offering 的 `$16.45` / `$28` 結構，推論籌碼已從「毒債壓盤」轉向「新機構 alignment」；2026-01-11 補充文則反駁 Andy Constan 的權證 / call 套利模型，認為核心問題在於 **ONDS** 期權流動性與實際執行摩擦。`$28` 是 common warrant exercise price，不是正式目標價，且 fully diluted shares / options hedge pressure 仍是核心追蹤項。

## 2026-01-11 補充：X 上權證套利模型爭議

| 爭點 | Andy Constan / 空方模型 | 大叔補充解讀 | 追蹤方式 |
|---|---|---|---|
| 權證 / call 價差 | 以約 `$1.22` warrant cost 對應 Jan 28 `$30` call 約 `$5.55` premium，理論上可鎖住價差 | 大叔認為這是 Excel 上漂亮，但真實市場需要足夠期權深度、borrow / short liquidity 與低摩擦執行 | options OI、bid/ask、成交量、borrow、short interest |
| 巨額部位執行 | 121M warrants / calls 語境下，可透過賣 call / short stock 壓制股價 | 大叔反駁 **ONDS** 期權市場容量不足，若巨量倒出 call，premium 會迅速崩塌，模型無法照理論價差執行 | call volume / premium collapse、dealer positioning、volume spikes |
| 資金用途 | 溢價入場主要是拿 cheap warrants，後續可能造成壓盤 | 大叔把 `$16.45` 資金視為 M&A、產能擴張與 defense platform buildout 的戰略彈藥 | 8-K / 10-Q use of proceeds、M&A closing、backlog、revenue / gross margin |
| CEO 公開回應 | 管理層反擊未必改變數學模型 | 大叔把 Eric Brock 公開反擊視為保護股價與自信訊號，並提醒 1/14、1/16 潛在催化 | 官方 PR、SEC filing、closing update、後續大叔文 |

## 需要避免的誤讀

| 項目 | 正確寫法 | 錯誤寫法 |
|---|---|---|
| `$28.00` | common warrant exercise price；大叔推論的新機構中期錨 | 大叔正式目標價、保證價、本專案目標價 |
| `$16.45` | Jan 2026 combined offering price / 新機構事件成本 | 絕對支撐或買點 |
| `$14.00` | 大叔原文圖中的事件現價 | 整理日現價或長期價值底 |
| 毒債清除 | outstanding convertible notes 已清償，有助移除舊 overhang | 公司完全沒有稀釋風險 |
| 無 reset | 大叔解讀 / 整理者在 424B5 快速檢索未找到 `reset` keyword | 完整法律保證或所有未來條款均無保護 |
