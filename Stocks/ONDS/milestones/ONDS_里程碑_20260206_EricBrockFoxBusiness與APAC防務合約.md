### 對 **ONDS** 的影響

大叔整理 Eric Brock 在 Fox Business 的訪談，重點是把同日官方宣布的亞太政府防務合約、Optimus Blue List 採購資格、critical infrastructure / C-UAS 需求，以及 NDAA / 非中國供應鏈合規，放進同一個 **ONDS** 成長敘事。

本篇升為 **ONDS** L4，原因是它不是單純 CEO 訪談：Brock 在訪談中提到公司當天宣布新的政府國防合約；Ondas 官方 2026-02-03 新聞稿確認該 classified defense contract 服務亞太政府客戶，初始系統預計 2026 年交付，並隨部署擴大延續交付。訪談同時把 2026-01-28 Optimus DCMA Blue List status 接到 rapid federal procurement，強化 **ONDS** 以合規自主無人系統切入 defense / homeland security / critical infrastructure 的路徑。

### 核心概念

- **Defense + homeland + critical infrastructure 同一條需求線**：Brock 把戰場、國土安全與關鍵基礎設施防護放在同一個無人系統需求週期中，呼應大叔此前的 C-UAS / airspace security thesis。
- **Blue List 是採購門檻，不是採購結果**：Optimus 取得 Blue List status 代表 federal procurement 摩擦下降，但仍要看 task order、部署與收入。
- **NDAA / 非中國供應鏈合規是護城河方向**：訪談提到 DJI / 中國供應鏈疑慮，大叔因此把 **ONDS** 放在合規自主無人系統供應商位置；真正驗證仍是政府採購與收入。
- **APAC contract 是國際 defense traction 訊號**：合約未披露金額，不能直接估值外推；但它把 **ONDS** 產品從歐美 / 中東 / 美國政策線延伸到亞太政府防務需求。
