# IBM 投資儀表板

> 最後更新：2026-04-29（補入 KP / FOMOSoc 2026-02-28 歷史 L2-L3：Claude Code / AWS Transform 與 legacy modernization 風險；不改變當前最新立場）

## 當前立場快照

- **綜合立場**：大叔觀點為隱含偏多 / 防禦 + 成長交集；**IBM** 不被寫成純 AI 高波動爆發股，而是高 recurring revenue、FCF、股息、混合雲與企業級 AI 變現的組合。
- **KOL 最新立場**：大叔 2026-04-23 長文認為 **IBM** 已從舊科技 / 低毛利 IT 服務印象，轉型成高毛利軟體、混合雲與企業級 AI 平台；`57.7%` 營業毛利率、`22.2 億美元` FCF 與 `45 億美元` 生成式 AI 訂單是核心證據。
- **本次補入的 KP 2026-02-28 legacy modernization 風險**：KP 認為 Claude Code / AWS Transform 類工具會讓 COBOL / mainframe 知識不對稱透明化，壓縮 IBM consulting 與 mainframe lock-in；但 code translation 不等於完整 modernization，zero downtime、governance、compliance 與 system integration 仍是 IBM 緩衝。此篇是 L2-L3 風險補強，不推翻大叔 2026-04-23 的 software / hybrid cloud / enterprise AI 現金流 thesis。
- **KP 歷史補強**：KP 2025-12-13 將 **IBM** 約 `$11B` 收購 Confluent 解讀為 hybrid cloud / enterprise AI strategy 的即時數據流補洞：OpenShift 是跨雲作業層，Confluent / Apache Kafka 是讓資料在企業各系統即時流動的神經系統。此篇補強 IBM AI infrastructure layer，但短期要追約 9x expected sales、EPS dilution 與 integration risk。
