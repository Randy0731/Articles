### 對 **VICR** 的影響

這篇將 **VICR** 建成正式 L3 主題個股文。大叔的核心判斷是：AI 算力的限制逐漸從「有沒有 GPU」推進到「能不能把高電流、低電壓、低損耗的電力穩定送到 GPU / package」。Vicor 的 FPA / vertical power / high-density module 若能靠近 advanced package 或被納入 GPU / AI server design-in，就可能從功率模組公司 re-rate 成 AI infrastructure power base。

語氣偏正面，但驗證壓力也很高。Q1 的 revenue `$113.0M`、GM `55.2%`、book-to-bill >2 與 backlog `$300.6M` 很強；但文章估值已假設 `$1.5B` annual capacity path 和 full-capacity EPS `$8-$10`。如果擴產、design-in 或 AI capex 任一環節延後，29x+ full-capacity P/E 仍可能被壓縮。

### 核心概念

- **AI 電力底座 / 垂直供電 re-rating**：rack power 越高，供電效率、current density、loss / heat control 越像算力利用率的決定因子。
- **封裝內供電 / 1.5mm design-in 護城河**：power module 一旦靠近 package，替換供應商會牽動 package layout、thermal、qualification 與系統可靠性。
- **SwaP 軍工底盤**：size、weight、power 對 aerospace / defense system 是硬限制；長 qualification cycle 可以提供 AI cycle 以外的需求底座。
