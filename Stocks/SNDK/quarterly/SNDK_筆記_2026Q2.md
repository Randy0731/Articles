## 本次結論

大叔把 **SNDK** fiscal Q3 解讀為 SanDisk 從 cyclical NAND / storage supplier 轉向 AI infrastructure storage capacity supplier 的拐點。official Q3 revenue `$5.950B`、non-GAAP gross margin `78.4%`、datacenter revenue `$1.467B` / q/q +233%、adjusted FCF `$2.955B` 與 FQ4 guide `$7.75B-$8.25B` revenue / `$30-$33` non-GAAP EPS，都支撐大叔的「cash flow machine」與 re-rating thesis。

個股追蹤上，**SNDK** 的重點不是單季 beat 本身，而是 NBM 能否把 memory cycle 的一部分轉成 multiyear / committed financials、enterprise SSD / TLC 是否真的因 KV cache / AI inference 取得 durable pricing power，以及 `78%+` gross margin 是否能在競爭者供給反應後維持。

## 大叔 2026-05-02 HBF follow-up

大叔把 Wccftech「NVIDIA not interested in HBF」的標題拆成 AI memory hierarchy 分流，而不是 **SNDK** thesis 失敗。HBF 在大叔框架中不是 HBM 的直接替代品，而是介於 HBM 與 SSD 之間的大容量樓層：HBM / SRAM 負責最熱、最低延遲的 compute path；HBF 可能放 model weights、long-context data 與 KV cache；SSD / NVMe 則保留更低成本的大容量 storage。

對 **SNDK** 的增量是，HBF optionality 從「概念性新樓層」變成需要逐季追蹤的 product / customer catalyst。SanDisk official HBF advisory board 與 fact sheet 支撐 HBF 技術方向存在；但 Google reportedly locked as first customer、NVIDIA not interested 等訊息仍來自 Wccftech 報導，不能寫成 Google / NVIDIA / SanDisk official 客戶公告。

| 面向 | 對 SNDK 的含義 |
|---|---|
| 立場 | HBF optionality 偏正面，但仍屬 L3 follow-up，不取代 NBM / datacenter enterprise SSD 這條已財報驗證的主 thesis |
| AI memory hierarchy | 若 HBF 進入 hyperscaler custom ASIC / TPU memory stack，SanDisk 可能從 storage capacity supplier 進一步被市場視為 AI memory layer supplier |
| Google reported customer | 對 re-rating 有潛在意義，但目前只能標媒體報導 / 未確認；需等 official confirmation、sampling、qualification 或 revenue evidence |
| NVIDIA non-interest | 大叔認為合理，不是 HBF 利空；NVIDIA training / HBM-centric stack 與 Google inference / capacity-tier stack 可同時存在 |
| 主要風險 | HBF bandwidth / latency / endurance / thermal、controller / software stack、advanced packaging yield、customer concentration 與 revenue timeline |
