## 2026-02-28 KP / FOMOSoc：HBF 記憶體新樓層

KP 的 HBF 前史很重要，因為它先把 **SNDK** 放進 AI memory hierarchy，而不是傳統 NAND cycle。KP 認為 HBF（High Bandwidth Flash）不是取代 HBM，而是在 HBM 與 SSD 之間增加一層高頻寬、高容量 Flash，用於大型模型權重、長上下文與 KV cache。

對 **SNDK** 的意義是：如果 AI inference 真的需要更多 intermediate memory / storage hierarchy，SanDisk 可能不只賣 commodity NAND，而是參與 AI server architecture。但這在 2026Q1 仍屬 product / standard optionality，成敗取決於 Nvidia design-in、OCP standard、SK Hynix partnership、CoWoS / packaging ecosystem、NAND pricing 與 AI server adoption。
