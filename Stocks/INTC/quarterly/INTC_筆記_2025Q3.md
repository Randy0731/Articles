### 對 **INTC** 的影響

KP 的主文是 **AMD** L3，但其中對 **INTC** 有明確歷史競爭風險：KP 將全球 x86 server CPU 市場描述為長期由 Intel Xeon 統治，並引用 PassMark 口徑稱 AMD EPYC 在 2025Q2 已達約 `50%` 份額，與 Intel 勢均力敵。這代表 Intel 數十年絕對壟斷被打破，是後續 **INTC** CPU 復權 thesis 的重要歷史壓力測試。

### 觀點摘要

| 面向 | 內容 |
|---|---|
| **立場** | 對 **INTC** 偏負面歷史風險；KP 認為 EPYC 憑性價比、核心密度與每瓦性能，正在從 Xeon 領地奪取份額 |
| **關鍵數字** | 全球 server CPU 市場約 `$22B`；EPYC 2025Q2 x86 server CPU share 約 `50%`，2024 年約 `30%`（KP / PassMark 口徑）；未提供 INTC 股價、目標價、買點或賣點 |
| **基本面** | Intel 若要在後續被重新定價，需要證明 Xeon roadmap、DCAI revenue、Foundry / 先進封裝與 AI 推論 CPU 復權能抵消 EPYC 壓力 |
| **催化劑 / 風險** | 後續追 DCAI revenue、Xeon / EPYC market share、perf-per-watt、TCO、gross margin、Diamond / Coral Rapids / 18A / 14A execution |

|---|---|---|---|---|
| EPYC 市占持續侵蝕 Xeon | 高 | AMD 在 hyperscalers / data center 持續擴大 share，Intel 無法用 roadmap / pricing / perf-per-watt 拉回 | server CPU market share、Xeon / EPYC benchmark、DCAI revenue、gross margin | KP 2025-08-09 |
| Intel CPU 復權 thesis 被競爭證偽 | 高 | Agentic AI 讓 CPU 需求上升，但主要受益者落在 **AMD** / **ARM**，而非 Intel | CPU attach ratio、customer wins、DCAI revenue、EPYC / ARM share | KP 2025-08-09；後續 Bytc / 大叔 INTC thesis |

### 對 **INTC** 的影響

KP 將 Intel 描述為高度二元的公司：未來不是巨大成功，就是極度沉淪。美國政府可能入股後，市場擔心補助轉股稀釋股東、公司未來仍需增發來支應龐大資金需求；KP 認為這些擔憂合理，但如果沒有政府安全網，Intel 作為美國本土先進製程希望的 foundry 部分更難取得足夠資本。

### 觀點摘要

| 面向 | 內容 |
|---|---|
| **立場** | 偏正面下檔保護，但不是無條件看多；KP 認為政府入股類似「Trump Put」，降低存續風險，但 upside 仍取決於商業執行 |
| **關鍵論點** | Foundry 是 Intel 最具戰略價值的部分；若分拆，foundry 失去 Intel 內部錨定客戶，設計部門也失去與製造工藝深度協同 |
| **股東風險** | 補助轉股可能稀釋既有股東，未來資金需求仍可能帶來更多股本發行；KP 沒有提供股價、目標價、買點或賣點 |
| **驗證問題** | Intel 能否在技術上追上 **TSM**、能否取得 **NVDA** 等外部大客戶、18A / Foundry / DCAI 是否轉成 revenue / margin / FCF |

### 風險表

| 風險 | 風險等級 | 觸發條件 | 觀察指標 | 出處 |
|---|---|---|---|---|
| 政府入股造成稀釋但未帶來商業成功 | 高 | 補助轉股 / 增發提高 share count，但 foundry 外部客戶、18A 良率或 FCF 未改善 | share count、government equity terms、cash burn、FCF、external foundry wins | KP 2025-08-23 |
| Foundry 分拆破壞設計製造協同 | 中高 | 市場或政策壓力推動分拆，使 foundry 失去內部錨定需求，design 也失去製程協同 | foundry customer mix、internal wafers、design roadmap、gross margin | KP 2025-08-23 |
| 國家隊保下檔但不保股東回報 | 高 | Intel 存續無虞，但技術追趕、外部客戶與 margin recovery 不及預期 | 18A yield、DCAI / Foundry revenue、external customers、gross margin、FCF | KP 2025-08-23 |
