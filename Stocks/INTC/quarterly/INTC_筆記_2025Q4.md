## KP / FOMOSoc：Intel 保留 NEX，從斷臂求生轉向系統整合豪賭

- **source_id**：`KP_FOMOSoc-20251206-kp-thinking-note-19-aws-google-mstr-robotics-nvidia-synopsys-intel-netflix-alab-crdo-2e8b672a`
- **來源**：KP / FOMOSoc Substack 週報
- **原文**：https://www.fomosoc.com/p/netflixcrdoalab-kp19
- **發文時間**：2025-12-06 03:43 UTC（台北 11:43）
- **整理日期**：2026-04-29
- **交會等級**：L3（已追蹤個股歷史補強）
- **主題 tags**：#AI基建 #ReRating #競爭風險 #執行風險 #政策風險

### 主旨

KP 在第 19 期將 Intel 取消 NEX spinoff 解讀為外部資金改善後的戰略反轉：原本 NEX 像是可出售換現金的非核心 network / edge business，但美國政府、SoftBank、Nvidia 合計約 `$15.9B` 資金進來後，Intel 有空間把 NEX 重新包裝成 Xeon CPU、networking、edge、software stack 與 Nvidia partnership 的 AI / data center system solution。對 **INTC**，這補強系統整合 optionality，但也提高「第三戰線」與執行分散風險。

### 個股觀點

| 面向 | KP 觀點 | 對 **INTC** 的資料庫含義 |
|---|---|---|
| 資產出售反轉 | 原本 Lip-Bu Tan 想拆分 / 出售 NEX 聚焦 CPU / Foundry，現在 Intel 宣布保留 | 與困境反轉 thesis 相接：資金壓力緩解後，公司從縮表求生轉向戰略保留 |
| NEX strategic role | NEX 年營收近 `$6B`，涵蓋 data center networking chips、telecom infra chips、edge compute platforms | 若能與 Xeon / AI server / edge software 打包，NEX 可成為 system integration 的一部分 |
| Nvidia partnership | Nvidia GPU 需要與 CPU / networking 低延遲交換，Intel 可嘗試用 Xeon + NEX networking 組合提供平台價值 | 延續 2025-09-20 Nvidia / Intel NVLink x86 CPU thesis，但仍需實際 design wins |
| Telecom / edge | Ericsson / Nokia 等 telecom infrastructure 依賴 Intel chips，保留 NEX 可維持 5G / edge end-to-end solution | 觀察 NEX 是否能帶來客戶黏性與 edge AI revenue，而非低 margin legacy business |
| 市場疑慮 | Intel 已同時打 Foundry vs TSMC、CPU vs AMD / ARM，保留 NEX 等於再多一條 Broadcom / networking 戰線 | 需提高管理層 focus、segment margin、FCF 與 capital allocation discipline 檢查權重 |

### 延伸追蹤

| 事件 | 追蹤重點 |
|---|---|
| 每季 Intel segment disclosure | NEX revenue / margin、edge / networking demand、是否仍有獨立披露或整併口徑 |
| Xeon + networking design wins | AI server architecture、Nvidia collaboration、CPU-network latency / bandwidth、customer adoption |
| Foundry / DCAI / NEX 資本配置 | Foundry losses、18A / external customers、DCAI share recovery、NEX capex / R&D allocation、FCF |
| 競爭對照 | Broadcom Tomahawk / networking share、AMD / ARM CPU competition、Ericsson / Nokia telecom demand |

### 風險與限制

- KP 的判斷是 Intel 有機會把 NEX 從非核心資產重新定義為 AI / edge / networking system piece，不是說 NEX 已證明高 ROI。
- 保留 NEX 會讓 Intel turnaround 的複雜度上升；若 Foundry、CPU、networking 同時分散管理層注意力，反而可能削弱執行紀律。
- Nvidia partnership 仍未等於 Intel Foundry 成功，也不等於 NEX 已取得 AI networking 設計勝利；需用財報與客戶證據驗證。
- 本文不是買賣建議。

## KP / FOMOSoc：Intel 止血成功，但還在復健室

- **source_id**：`KP_FOMOSoc-20251025-kp-thinking-note-13-openai-gm-quantum-aws-tsla-intc-nflx-bd0ee08c`
- **來源**：KP / FOMOSoc Substack 週報

- **發文時間**：2025-10-25 02:30 UTC（台北 10:30）
- **整理日期**：2026-04-29
- **交會等級**：L3（已追蹤個股歷史補強）
- **主題 tags**：#財報 #AI基建 #ReRating #競爭風險 #執行風險 #政策風險

### 主旨

KP 將 Intel 最新財報比喻為 ICU 病情報告：公司結束連續六季虧損、重返獲利，代表止血成功；但 data center share loss 與 Foundry 虧損仍是慢性病。此篇延續 KP 先前「Trump Put」與 Nvidia-Intel 合作框架，補上 2025Q4 的困境反轉觀點。

### 個股觀點

| 面向 | KP 觀點 | 對 **INTC** 的資料庫含義 |
|---|---|---|
| 止血成功 | Intel 結束連續六季虧損，PC demand 回暖，新 CEO Lip-Bu Tan 的成本紀律奏效 | 補強 turnaround relief，但需分辨節流 vs 真正造血 |
| 外部強心針 | 美國政府入股、Nvidia 採購 / 投資 CPU、SoftBank 入股共同重建信心 | 延續國家隊 Put / Nvidia platform adjacency，但不等於 Foundry 成功 |
| Data center 慢性病 | Data center 曾是金雞母，仍被 AMD / ARM 蠶食，年比營收仍弱 | 延續 DCAI / server CPU share 檢查 |
| Foundry 慢性病 | Foundry 本季虧損約 `$2.3B`，外部客戶不足前仍會拖累整體盈利 | 延續 18A、external foundry wins、utilization、gross margin、FCF 檢查 |
| AI 後勤總管 | 推理時代需要低成本高效率 CPU，Intel 的 Xeon 可作 AI data center 後勤貨車 | 與 Bytc / 大叔 2026 CPU 復權觀點形成歷史呼應 |

### 延伸追蹤

| 事件 | 追蹤重點 |
|---|---|
| 每季財報 | DCAI revenue、Xeon backlog、server CPU share、gross margin、cost discipline、EPS quality |
| Foundry updates | Foundry external customers、18A yield、advanced packaging orders、Foundry operating loss、utilization |
| Strategic backing | Government equity terms、Nvidia CPU roadmap / orders、SoftBank stake, dilution / governance impact |

### 風險與限制

- KP 口吻明確偏正面，稱 **INTC** 是他少數較明確看好的股票之一；入庫時仍保留「上檔靠商業執行」的限制，不寫成買賣建議。
- 若盈利主要來自成本削減，而非 DCAI / Foundry 真正改善，turnaround 可能只是 relief rally。
- Foundry 虧損與 AMD / ARM 競爭仍是核心風險。
