### 主旨

KP 將 Intel 最新財報比喻為 ICU 病情報告：公司結束連續六季虧損、重返獲利，代表止血成功；但 data center share loss 與 Foundry 虧損仍是慢性病。此篇延續 KP 先前「Trump Put」與 Nvidia-Intel 合作框架，補上 2025Q4 的困境反轉觀點。

### 個股觀點

| 面向 | KP 觀點 | 對 **INTC** 的資料庫含義 |
|---|---|---|
| 止血成功 | Intel 結束連續六季虧損，PC demand 回暖，新 CEO Lip-Bu Tan 的成本紀律奏效 | 補強 turnaround relief，但需分辨節流 vs 真正造血 |
| 外部強心針 | 美國政府入股、Nvidia 採購 / 投資 CPU、SoftBank 入股共同重建信心 | 延續國家隊 Put / Nvidia platform adjacency，但不等於 Foundry 成功 |
| Data center 慢性病 | Data center 曾是金雞母，仍被 AMD / ARM 蠶食，年比營收仍弱 | 延續 DCAI / server CPU share 檢查 |
| Foundry 慢性病 | Foundry 本季虧損約 `$2.3B`，外部客戶不足前仍會拖累整體盈利 | 延續 18A、external foundry wins、utilization、gross margin、FCF 檢查 |
| AI 後勤總管 | 推理時代需要低成本高效率 CPU，Intel 的 Xeon 可作 AI data center 後勤貨車 | 與 Bytc / 大叔 2026 CPU 復權觀點形成歷史呼應 |

### 延伸追蹤

| 事件 | 追蹤重點 |
|---|---|
| 每季財報 | DCAI revenue、Xeon backlog、server CPU share、gross margin、cost discipline、EPS quality |
| Foundry updates | Foundry external customers、18A yield、advanced packaging orders、Foundry operating loss、utilization |
| Strategic backing | Government equity terms、Nvidia CPU roadmap / orders、SoftBank stake, dilution / governance impact |

### 風險與限制

- KP 口吻明確偏正面，稱 **INTC** 是他少數較明確看好的股票之一；入庫時仍保留「上檔靠商業執行」的限制，不寫成買賣建議。
- 若盈利主要來自成本削減，而非 DCAI / Foundry 真正改善，turnaround 可能只是 relief rally。
- Foundry 虧損與 AMD / ARM 競爭仍是核心風險。
