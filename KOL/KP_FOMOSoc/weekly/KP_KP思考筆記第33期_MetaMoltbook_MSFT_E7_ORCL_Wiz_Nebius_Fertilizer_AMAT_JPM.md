## 0. 反脆弱配置：不操作，也可以是高品質作為

KP 開場延續第32期的反脆弱配置：戰爭與市場不確定性高時，最誠實的答案常常是承認自己不知道局勢會如何演變。Howard Marks 的重點不是躺平，而是當資訊不足時，不要硬把模糊事件翻成交易動作。

Buffett 的棒球理論被用來補充這個框架：投資沒有三振出局，投資人不需要對每一顆球揮棒。真正的作為是場邊研究、拆基本面、等甜蜜點出現。整理者判定：這段延續第32期「反脆弱組合 / 多護城河框架」，不新增單股觀點。

## 1. Meta 收購 Moltbook：AI agent 社交圈的藍圖

Moltbook 是 AI agent 版 Reddit：人類不能參與，AI 代理人在上面發文、辯論、分享程式碼。它同時充滿泡沫與漏洞：大量內容可能是人類提詞的 AI 劇場，資料庫外洩顯示真實人類擁有者約 1.7 萬、與百萬級 AI 居民形成 88:1 落差，身分偽裝也很容易。

KP 認為 Meta 出手不是因為 Moltbook 已經是成熟產品，而是因為它碰到下一代社交基礎設施：AI agent 的可驗證身分、Agent-to-Agent 通訊層，以及能觀察 AI 自主發現、談判、協作的實驗場。Meta 買的是藍圖與建築師，也是在 OpenAI 收走 OpenClaw 創辦人後的人才防禦。

整理者判定：**META** 既有 watchlist 補強。這與第23期 Manus execution layer、第20期 Avocado / AI ads 共同形成 Meta agent infrastructure / AI ads automation 拼圖。後續追 Moltbook team integration、MSL roadmap、agent identity、A2A protocol、AI social graph、Meta AI ads / commerce use case 與安全治理。

## 2. Microsoft E7：把 AI agent 當員工管理

Microsoft 365 E7（The Frontier Suite）被 KP 視為 E5 以來最大的企業授權進化。E5 是安全與合規城牆，E7 則是 AI 指揮部：不只是讓人安全工作，而是讓 AI agent 替人執行任務。

核心是 Agent 365、Work IQ、Entra Suite 與 Copilot Wave 3。Agent 365 讓企業像管理員工一樣管理 AI 代理人的身分、權限與行為；Work IQ 把企業文件、會議與決策語境變成集體記憶；Entra 把 AI agent 納入身分治理。價格每人每月 `$99`，比 E5 高約 65%，但若拆買 Copilot `$30`、Agent 365 `$15`、Entra Suite `$12`，總價可能超過 `$115`，E7 用一站式打包換整合省心。

整理者判定：**MSFT** 既有 watchlist 補強。本段延續第32期 Token 節流閥 / AI 軟體治理框架，但更進一步變成「AI 員工治理 / Work IQ lock-in 框架」。後續追 E7 adoption、Agent 365 功能落地、Work IQ lock-in、Copilot gross margin、non-OpenAI model routing、Claude / Anthropic support、enterprise AI ROI 與每席 `$99` 是否能被 CFO 接受。

## 3. Oracle：用客戶資產負債表蓋 AI data center

KP 將 Oracle Q3 前的市場焦慮總結為「錢從哪裡來」：每年約 `$50B` capex、FCF 負 `$24.7B`、債務超過 `$120B`。Oracle 的回答是 BYOG（Bring Your Own GPU）與客戶預付款：客戶自備昂貴 GPU，Oracle 只蓋機房、電力與冷卻，並用預收款提高經營現金流。

管理層稱新增 AI 合約不會造成負現金流壓力，CFO Doug Kehring 也表示 2026 年餘下時間無需再進入債券市場融資。RPO 達 `$553B`，且部分訂單伴隨預付款或客戶硬體，使毀約機率下降。這把 Oracle 從高風險賭徒改寫成資源調度者。

但風險沒有消失。BYOG 讓 Oracle 管理各式非標準硬體，增加 opex 與優化難度；硬體由客戶持有，Oracle 無法賺完整硬體轉租溢價，只能賺機房、電力、冷卻與軟體服務費；若客戶 GPU 供應鏈延遲，機房會空轉。整理者判定：**ORCL** 既有 watchlist L3 補強。後續追 OCI revenue / RPO conversion、customer prepayment、BYOG hardware delivery、gross margin 是否維持 32%+、capex / FCF、debt market access 與 mixed-hardware operations。

## 4. Google 收購 Wiz：雲端安全成為 Cloud 戰爭門票

Google 以 `$32B` 全現金收購 Wiz，是 Google 史上最大併購。KP 認為這不是一般安全產品補強，而是 Google Cloud 的戰略補課。雲端共同責任模型下，雲商負責底層安全，但客戶仍要保護自己的資料、應用、身分與設定；若這件事太複雜，客戶就不敢把核心業務與 AI workload 上雲。

Google 既有 Mandiant、Chronicle 偏向威脅情報與事後分析，缺的是事前預防、多雲、無代理、風險排序的 CNAPP。Wiz 則是這個缺口裡增長最快的明星。AWS 有 GuardDuty、Inspector、Security Hub，Microsoft 有 Defender for Cloud、Sentinel、Entra；Google 選擇用巨額收購直接追趕。

整理者判定：**GOOG**/**GOOGL** 為正式追蹤個股 L3，同步 `Stocks/GOOGL/`。這補上 Google Cloud enterprise trust / AI security layer，但仍有風險：Google 承諾保持 Wiz 品牌獨立與多雲支持，需避免客戶因 vendor lock-in 疑慮轉向 Orca、Prisma Cloud、CrowdStrike 等替代；`$1B` 留任獎金也顯示人才留存是硬考題。後續追 deal close、regulatory review、Wiz ARR / growth、Google Cloud security revenue、multi-cloud retention、Wiz integration、talent retention、CNAPP win rate 與 margin。

## 5. Nebius / Nvidia：AI 基建循環經濟的再押注

Nvidia 投資 Nebius `$2B`，KP 視為關係從客戶升級成深度戰略同盟。Nvidia 早在 2024 年底已投入約 `$33M`，這次持股從約 0.5% 提升到 8.3%。邏輯類似 CoreWeave：Nvidia 投資 AI cloud 客戶，客戶拿到資金與背書，再買下一代 Nvidia 晶片。

這筆錢緩解 Nebius 的短期資金焦慮。Nebius 現金約 `$3.7B`，但 2026 capex 計畫高達 `$16B-$20B`，`$2B` 可覆蓋約 10-12% 缺口。更重要的是 GPU supply 優先權、Rubin / Vera 設計支持、Nvidia CEO 背書，以及 AI 工廠設計、軟體 stack 與 GPU health monitoring 的 co-engineering。

整理者判定：**NBIS** 既有 watchlist L3 補強，**NVDA** 為生態通路 context。關鍵風險從「能否活下去」轉為「能否兌現承諾」：全球 9 個 data centers、超過 `$20B` backlog 轉收入、2026 年底 800MW-1GW online capacity、40% adjusted EBITDA margin 與 FCF 轉正，都還要驗證。單一路線深綁 Nvidia 也是雙面刃，若 AMD 或自研 ASIC 更有吸引力，Nebius 彈性會下降。

## 6. 戰爭與肥料：油價只是煙霧彈

KP 認為 Hormuz 危機若只看油價會誤判。金融市場會提前反映 reopening / escort 的好消息，原油期貨可能幾天內從高點回落 30-60%；但實體供應鏈修復至少需要 1-2 個月，油田重啟、油輪排程、煉化與運輸都存在時間差。

更深層風險是肥料。波斯灣地區是全球化肥工廠，KP 口徑全球海運貿易中 30-50% 尿素、30% 氨與大量磷 / 硫來自此區。不同於石油，全球沒有戰略化肥儲備；每月約 133 萬噸肥料經過 Hormuz，且 100% 依賴大型散裝船，沒有尿素管道或氨氣管道可繞道。若船流從每日 150 艘降到 13 艘，肥料會卡在倉庫。

本次危機還碰上北半球春季播種窗口。即使工廠 1-2 週能重啟、貨船快速離港，農民錯過施肥時間也無法回補，2026 秋季收成與未來一年食品價格才是真正結果。整理者判定：新增「化肥物理瓶頸 / 糧食危機滯後框架」。本文未具名北美化肥 ticker，不建立 **CF** / **NTR** / **MOS** 索引觀點。

## 7. AMAT EPIC：設備商變成記憶體創新中樞

AMAT 宣布美光與 SK 海力士加入其矽谷 EPIC 中心成為創始夥伴，加上 2 月已加入的 Samsung，全球記憶體三巨頭在同一研發平台匯合。KP 將 EPIC 稱為「研發共享廚房」：AMAT 把客戶工程師嵌入自己的研發體系，讓設備、材料、製程與 HBM / NAND 問題同步驗證。

EPIC 的目標是縮短約 30% 商用化時間。與合資研發中心不同，EPIC 由 AMAT 100% 獨資興建與營運，Samsung、Micron、SK Hynix 不持股也不出資。這讓 AMAT 成為規則制定者與技術緩衝區，並掌握三大記憶體廠未來路線圖和核心痛點。

整理者判定：**AMAT** 既有 watchlist L3 補強，**MU**、**000660.KS**、**005930.KS** 為 memory customer context。這新增「Lab-to-Fab / 半導體設備共同設計框架」：設備商不只是賣工具，而是成為 AI 硬體架構共同設計者。後續追 EPIC partner projects、HBM4 / DRAM / NAND process wins、ALD / advanced packaging tools、service revenue、orders / backlog、gross margin、China exposure 與 customer concentration。

## 8. JPM / 私人信貸：Back leverage 的慢火去槓桿

JPMorgan 開始調降部分私人信貸貸款抵押品估值，KP 視為 private credit 壓力的關鍵一刀。私人信貸基金過去用 back leverage 放大收益：拿貸款組合向銀行抵押借錢，將 8% 原始收益放大成雙位數回報。當銀行認定抵押品不如以前值錢，基金可借額度下降，若跌破門檻還可能被 margin call。

KP 不把這寫成 2008 重演。今日 private credit 近 `$2T`，占全球證券總額不到 1%；2008 次貸相關證券約占 5%。私人信貸多是點對點貸款，風險封閉在基金內，且 redemption gates 可放慢擠兌。銀行經歷 2008 後資本更充足，JPM 的行動更像主動風控而非恐慌自保。

但慢火風險仍大。銀行對私人信貸總曝險近 `$300B`；未來 2026-2027 有超過 `$1T` 私人貸款到期，融資利差可能增加 200-400 bps。半開放式基金、BDCs 與過度依賴槓桿者承壓；dry powder 充足、封閉式結構穩健者與 distressed / special situations 可能受益。整理者判定：**JPM** 既有 watchlist L3 補強，**BX**、**KKR** 作 private credit fund manager context。這與第32期 BlackRock gating、第30期 Blue Owl 半流動產品錯配連成同一條 private credit 去槓桿鏈。
