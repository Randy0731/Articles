## 主旨

Bytc 用「便當漲 10 元」拆解油價如何沿著供應鏈傳導到日常物價：原油被煉成汽油、柴油、石化原料、塑膠、化肥、農藥、潤滑油與瓦斯，再透過物流、包材、農業成本、餐飲燃料、租金與薪資，逐層滲入一個便當的成本。

文章不是單純看多油價，而是建立一個通膨與資產配置框架：油價短期上漲先壓物流與能源成本，3-6 個月後進入農業與包材，6-12 個月後才反映在薪資、租金、菜單價格與 CPI 黏性。若通膨停留在 2.5%-4% 這種溫和但持久的區間，它會懲罰現金與固定票息債券，尤其是長天期國債；相對保護有定價權的股票、實物資產與能把成本轉嫁的企業。

## 數字與時間線

| 時間 / 數字 | Bytc 口徑 | 解讀 |
|---|---|---|
| Hormuz 海峽 | 約全球石油貿易 20% 經過，最窄處約 33 公里 | 地緣風險會立刻進入油價風險溢價 |
| 油價反應 | 中東緊張時，原油價格可能在數小時內上漲 3%-5% | 第一波是 risk premium，不一定先看到實體短缺 |
| 柴油成本 | 卡車營運成本中燃料可達約 30%-40%，柴油 +10% 可能讓整體車隊成本 +3%-4% | 物流價格是第一波傳導 |
| 化肥 | 稻作每公頃可能用 100-150 kg 尿素；2021-2022 歐洲肥料價格一度翻倍 | 能源價格會透過化肥影響糧食與餐飲 |
| 餐飲瓦斯 | LPG / 天然氣可占便當店營運成本約 5%-10% | 小店利潤薄，能源成本上升容易進入菜單 |
| 1-4 週 | 物流、航空、航運與電費燃料附加費先反應 | 第一波：可見成本 |
| 3-6 個月 | 化肥、包材、橡膠、塑膠與農產品報價調整 | 第二波：中游成本 |
| 6-12 個月以上 | 薪資、租金、CPI 權重、消費者預期與菜單價格黏性 | 第三波：通膨心理與制度化 |
| 1970s vs now | 1970s 私部門工會率約 25%，現已低於 6%；美國能源效率與能源地位也不同 | 這次更像溫和但黏性的成本通膨，而非完全複製 1970s |
| 3% 通膨十年 | 購買力約剩 74% | 現金購買力慢性流失 |
| 美國國債 | Bytc 文中寫美國聯邦債務已超過 `$39T` | 通膨可提高名目 GDP 分母，稀釋舊債務負擔 |
| 二戰後 | 美國 debt/GDP 從約 120% 降到約 30% | 成長 + 溫和通膨 + 金融壓抑是歷史債務出清路徑 |

## 核心框架 / 心法

- 油價不是單一商品價格，而是一條從能源、石化、農業、物流到消費端的成本鏈。
- 便當漲價不是孤立事件，而是柴油、化肥、包材、瓦斯、人工、租金與預期一起疊加後的末端表現。
- 溫和通膨會讓「看起來安全」的現金與長債變成購買力流失來源；固定票息若低於實際通膨，名義上沒虧，實質上被稀釋。
- 股市在溫和通膨下的勝負不在於是否完全避開通膨，而在於企業能否轉嫁成本、保住毛利，甚至利用景氣波動整合弱者。
- 對政府來說，通膨是一把不會公開承認、但歷史上常用的債務稀釋工具。

## 風險與反例

| 風險 / 反例 | 判讀 |
|---|---|
| 油價衝擊若變成供給中斷與工資螺旋 | 會從溫和通膨升級成更危險的停滯性通膨，需要重新評估 Fed 反應 |
| 美元走強或需求下滑 | 可能壓制油價與 commodity pass-through，讓通膨傳導中斷 |
| 企業定價權被高估 | 若消費者抗拒漲價，企業可能只剩毛利壓縮，而不是成功轉嫁 |
| 長債殖利率已重新定價 | **TLT** 風險不等於永遠不可碰；若通膨下行與降息同時發生，duration 反而可能受益 |
| 通膨稀釋債務也會傷害民生 | 對政府是分母工程，對家庭則是購買力壓力；不能只看宏觀漂亮帳 |
