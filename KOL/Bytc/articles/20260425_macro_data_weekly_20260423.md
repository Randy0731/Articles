## 主旨

Bytc 將本週盤面概括為「價格在樂觀，主力在調倉」。指數創高、科技股領漲、股票基金與 sector funds 大幅流入，CTA 連續三週提供買盤，企業 buyback authorization 也創高；但 VIX 反而升到 18.70、breadth 降到 2023 年中以來低位、pension rebalancing 下週可能帶來約 230 億美元調倉，代表短線追價與結構集中風險都在升高。

他的中長期判斷仍偏多：美股 breakout、企業回購、CTA 仍有 up-tape 買盤、AI 基建與供應鏈瓶頸吸走科技 / 工業 / 金屬資金。但短線因三週暴力反彈、CTA 接近 full、market breadth 不佳，他準備在下週修剪部位，把現金提高到 20%-25%。這是 KOL 個人倉位紀錄，不是本專案建議。

本文的公司段落有三個增量：**TSLA** 從交車數字轉向 FSD / Robotaxi / Optimus / 自研 edge chips；**INTC** 被放進 AI Agent 推理時代 CPU 配比重估，並觸發本 repo 已追蹤個股同步；**NOW** / software 估值則被 Bytc 明顯下修信心，因 AI giants、算力成本與 enterprise internal agents 可能壓縮傳統 SaaS 中間層。

## 核心框架 / 心法

- **價格樂觀、主力調倉**：指數高點與 fund flows 偏多不代表無風險。當 VIX 上升、Put/Call 下滑、breadth 變差、pension rebalancing 將至，Bytc 會把「中長期 bullish」與「短線提高現金」分開處理。
- **AI physical bottleneck 輪動**：科技、工業、金屬與礦業同步流入，代表市場不只買軟體想像，而是在買 AI compute 受限於晶片、電力、材料、供應鏈與先進製造的真實瓶頸。
- **Agentic AI CPU 復權**：在 AI Agent inference 裡，CPU 不只是 GPU 的低價配角；若 CPU:GPU 配比從 1:8 走向 1:4 甚至 1:1，**INTC**、**AMD**、**ARM** 都可能因伺服器架構重估受益。
- **SaaS 中間層壓縮**：AI giants 具備頂尖軟體人才與模型 / compute stack，企業也可能用 internal agents 繞過部分 SaaS middleware。傳統 software 的防守點是 compliance、workflow stability、audit trail 與 hallucination responsibility，但這比較像防守，不一定支撐高成長 multiple。
- **RWA stablecoin 選邊**：Ondo / OUSG / USDY 的核心不是 crypto project competition，而是傳統金融巨頭用合規、分層產品、海外非美國人市場與 DeFi collateral integration 重新設計 on-chain dollar distribution。
