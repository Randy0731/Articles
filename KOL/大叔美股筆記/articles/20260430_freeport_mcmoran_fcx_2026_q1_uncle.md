## 摘要

大叔將 **FCX** 2026Q1 財報定義為「價格勝過產量」的典型案例：Grasberg Block Cave 因 wet muck / mud-rush 後續工程而降載，短期產量受壓；但銅價、金價與鉬價同步走高，使 FCX 在低產量下仍交出 revenue、net income 與 adjusted EPS beat。大叔的核心判斷是，不應把短期產能下修直接解讀為衰退；在 AI data center、EV 與全球電網升級共同推升銅需求，而供給彈性不足的背景下，FCX 更像銅價與長期電氣化需求的現金流放大器。

官方 Q1 release 校準：2026Q1 revenue `$6.234B`、net income attributable to common stock `$881M` / `$0.61` per share、adjusted EPS `$0.57`；consolidated production 為 copper `662M lbs`、gold `97k oz`、molybdenum `22M lbs`；average realized copper price `$5.78/lb`。FCX 也宣布與印尼政府簽署 Grasberg life-of-resource extension MoU，並調整 Grasberg Block Cave ramp-up，預期 2026H2 約 65% capacity、2027 mid-year 約 80%、2027 年底接近 full capacity。

## 核心 thesis

大叔的主軸不是「Grasberg 產量少了所以 FCX 變差」，而是「在供給受限時，價格主導產業邏輯」。Q1 顯示銅價 `$5.78/lb` 已足以抵消部分產量下滑，且 gold / molybdenum by-product credits 使 unit cost 更具彈性。

大叔把 FCX 的長期敘事放在三條線上：

- **AI / 電網 / EV 對銅需求的高確定性**：算力與用電需求本質上需要實體銅，而不是只有 GPU / HBM / 光通訊。
- **供給彈性不足**：礦山開發、品位下降、capex 壓力與環境 / 地緣風險使新增供給跟不上。
- **Grasberg 風險與 terminal value 重估同時存在**：短期 wet muck / loading bottleneck 是壓力；life-of-resource MoU、El Abra expansion、Arizona leaching 與自動化則是長線 optionality。

大叔一句話總結是：買 FCX 是買未來十年全球用電與算力的物理需求。此句保留為 KOL 觀點，不是本專案買賣建議。

## Grasberg wet muck 與工程解法

大叔把 Grasberg Block Cave 的 wet muck 問題視為「安全優先 + 工程升級」，而不是單純營運失敗。文章拆成三層：

| 層次 | 大叔整理 | 投資含義 |
|---|---|---|
| Open-pit bottom dewatering | 舊露天坑底部大量排水 / 排泥，降低濕黏土進入地下 drawpoints 的壓力 | 先止血，避免濕礦料繼續放大安全風險。 |
| Drainage gallery | 新增約 3 km drainage tunnel / gallery，2026H2 drilling，2027 完整排水 | 2027 是否如期完成，是大叔定義的主要重估時間窗。 |
| Automation / remote operation | wet muck control chute、remote loading、remote rock breakers、autonomous trains | 若完成，Grasberg 可從事故修復轉成更安全 / 自動化的長壽命資產。 |

官方 release 也確認：PTFI 需修改 ore-loading infrastructure，當前 bottlenecks 預計可在 2027 年中大致處理，production rates 2026H2 約 65%、2027 年中約 80%、2027 年底接近 full capacity。
