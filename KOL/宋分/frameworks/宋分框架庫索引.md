## DCF 收購價值 / PE 轉售市場 re-rating 框架

- **來源**：`KOL/宋分/articles/20260224_dcf_pe_target_rerating_resale_market.md`
- **一句話定義**：DCF 回答整家公司長期能產生多少現金流與 private market 收購價值，PE 回答 public market 在某個時間點願意用多少倍數轉售股票；成長股大行情常來自市場改變評價方法。
- **適用情境**：DCF 與 PE target 差距很大、成長股結構轉折、sellside target 高於 DCF、CAPEX-heavy 公司從投資期走向平台期、想分辨 intrinsic value / downside protection 與 exit multiple / re-rating 的情境。
- **觀察指標**：intrinsic value、private market price、public market PE、exit multiple、FCF、EV coverage、capex intensity、gross margin normalization、operating leverage、EPS at exit、market narrative shift、valuation method change。
- **相關文章**：宋分-20260224-dcf-pe-target-rerating-resale-market-d1cf3e16
- **可套用 ticker / 產業**：無具名 ticker；成長型科技股、AI / data center 擴產公司、CAPEX-heavy 公司、從投資期轉向平台期的結構轉折標的。

## PEG 成長錯估 / 信任不足框架

- **來源**：`KOL/宋分/articles/20260219_peg_market_decoding_growth_mispricing.md`；`KOL/宋分/articles/20260223_nvda_earnings_preview_ai_duration_visibility.md`
- **一句話定義**：PEG 低不是便宜結論，而是市場對成長持續性信任不足的訊號；只有先獨立判斷公司具備多年成長，低 PEG 才可能代表市場把成長股誤當景氣股定價。
- **適用情境**：財報好但股價不漲、EPS 連續上修、Forward PE 下降但市場仍懷疑、成長股從一次性改善轉向多年成長、景氣循環股低 PEG 風險辨識；**NVDA** PEG 0.37 這類 AI capex duration 爭議。
- **觀察指標**：Forward PE、next-year EPS growth、PEG、multi-year EPS revision、growth duration、PE re-rating、cyclical peak vs sustainable growth、consensus belief、target multiple、sell-side revision、2027 visibility、order duration。
- **相關文章**：宋分-20260219-peg-market-decoding-growth-mispricing-ac62ac3f；宋分-20260223-nvda-earnings-preview-ai-duration-visibility-d6cf0c58
- **可套用 ticker / 產業**：**NVDA**；成長型科技股、AI / data center、EPS 連續上修但市場尚未完全相信的標的；原物料股、航運股、記憶體僅作景氣循環反例，需避免把低 PEG 直接寫成便宜。

## 確定性提升 / 市場信任建立框架

- **來源**：`KOL/宋分/articles/20260221_investment_discipline_3_certainty_repricing.md`
- **一句話定義**：股價加速常來自市場不再那麼懷疑，而不是公司第一次變好；報酬集中在公司證明、產業證明或 market regime 改變使信任跨過門檻後。
- **適用情境**：看對公司但長期盤整、賣掉後大漲、財報改善但市場仍懷疑、一次性好財報 vs 可重複改善、成長股 re-rating 初期。
- **觀察指標**：company confirmation、industry confirmation、market regime、repeatability、連續季度 revenue / margin / FCF 改善、guidance 上修且達成、peer results、upstream orders、customer demand、rates、liquidity、risk appetite、valuation multiple。
- **相關文章**：宋分-20260221-investment-discipline-3-certainty-repricing-ad32a16d
- **可套用 ticker / 產業**：無具名 ticker；成長型科技股、turnaround、景氣循環轉折、產業需求復甦、利率 / 流動性 regime 變化下的 re-rating 標的。

## 分析師流程 / 事件等待框架

- **來源**：`KOL/宋分/articles/20260211_investment_discipline_analyst_process.md`；`KOL/宋分/articles/20260215_investment_discipline_2_research_order.md`；`KOL/宋分/articles/20260221_investment_discipline_3_certainty_repricing.md`；`KOL/宋分/articles/20260225_amd_meta_6gw_eps_model_process.md`
- **一句話定義**：先看公司與未來事件，再看價格；研究不應從看盤、K 線、新聞與題材開始，且要判斷市場正在等待哪個可驗證訊號才會提高確定性；具名事件要再往下拆成 revenue、net income、EPS 與 multiple。
- **適用情境**：財報後急跌、長期盤整、短線反彈後想追、投資人想分辨跌價是基本面破壞還是情緒 / 估值壓縮、想把 PE / DCF 作為情緒校準而不是股價預測、或想判斷市場何時開始相信公司變好的情境；訂單 / 合作新聞需要進模型的情境。
- **觀察指標**：thesis 是否變壞、研究順序、是否反向合理化價格、新產品、訂單回升、產能開出、成本下降、EPS / revenue revision、Forward PE、DCF assumptions、guidance、company confirmation、industry confirmation、market regime、fund flows、盤整期間的成交量與估值區間、revenue recognition、net margin、diluted shares、PE multiple。
- **相關文章**：宋分-20260211-investment-discipline-analyst-process-16636bb7；宋分-20260215-investment-discipline-2-research-order-da28e1fb；宋分-20260221-investment-discipline-3-certainty-repricing-ad32a16d；宋分-20260225-amd-meta-6gw-eps-model-process-f913e9c3

## 共識變化 / 預期差定價框架

- **來源**：`KOL/宋分/articles/20260211_consensus_expectation_gap_market_pricing.md`；`KOL/宋分/articles/20260221_investment_discipline_3_certainty_repricing.md`
- **一句話定義**：市場形成報酬的關鍵不是公司當下好壞，而是市場共識與確定性何時改變；報酬通常出現在信任尚未完全建立、但可重複證據正在累積的階段。
- **適用情境**：看對公司但賺不到、好公司長期盤整、財報好股價不動、壞消息公布後反彈、想判斷市場何時開始定價已知事實或從懷疑轉向相信的情境。
- **觀察指標**：consensus revision、EPS / revenue revision、sell-side tone、news sentiment、company confirmation、industry confirmation、market regime、price reaction vs fundamentals、fund flows、institutional positioning、volume、valuation multiple、是否仍有新買盤理由。

## 評價溫度計框架

- **來源**：`KOL/宋分/articles/20260205_amd_valuation_thermometer.md`；`KOL/宋分/articles/20260205_avgo_valuation_thermometer.md`；`KOL/宋分/articles/20260205_nvda_valuation_thermometer.md`；`KOL/宋分/articles/20260206_amzn_capex_cycle3_valuation_thermometer.md`；`KOL/宋分/articles/20260207_forward_pe_market_decoding.md`
- **一句話定義**：把一檔股票放在「最便宜」到「最貴」的主觀區間上，依歷史評價區間、成長確定性、現金流穩定度、市場預期位置與公司自身資本配置判斷現在更靠近地板或天花板。
- **適用情境**：財報後急跌、新聞疑慮、產品延後、CAPEX / FCF 恐慌或內部人 / 公司資本配置造成股價回落，但基本面未明顯惡化且估值快速壓縮的高品質成長股；尤其適合 AI 硬體 / 半導體 / hyperscaler / mega-cap growth 在市場期待差極高時的檢查。
- **觀察指標**：forward PE / EPS consensus、公司回購價、FCF growth、AI revenue / data center revenue、ASIC / GPU / CPU / AWS 產品節點、gross margin guidance、內部人交易、訂單 / backlog 與市場 consensus gap、H200 / Rubin / MI450 / custom ASIC / AWS backlog 節點、市場是否已經 price in 過高期待；便宜原因是衰退還是擴大投資。
- **相關文章**：宋分-20260205-amd-valuation-thermometer-6dd1d930；宋分-20260205-avgo-valuation-thermometer-d7fe5606；宋分-20260205-nvda-valuation-thermometer-3ad8a4bd；宋分-20260206-amzn-capex-cycle3-valuation-thermometer-a097e225；宋分-20260207-forward-pe-market-decoding-c6a2a228
- **可套用 ticker / 產業**：**AMD**, **AVGO**, **NVDA**, **AMZN**；AI accelerator、server CPU / GPU、custom ASIC、inference / training、hyperscaler CAPEX、AWS / cloud infrastructure、半導體成長股、財報後 re-rating 標的。

## Forward PE 悲觀錯價框架

- **來源**：`KOL/宋分/articles/20260207_forward_pe_market_decoding.md`；`KOL/宋分/articles/20260209_dcf_buyside_valuation_process.md`
- **一句話定義**：用明年或更遠 EPS 看現在股價；當今年 EPS 因擴產、AI 投資或資料中心建置被下修、股價先跌時，若未來需求未消失，Forward PE 可能先顯示市場過度悲觀造成的錯價。
- **適用情境**：成長型科技股、AI / data center 擴產期、CAPEX / product-cycle transition、新聞與近端 EPS 悲觀但長期需求仍可驗證的情境；也可作評價溫度計的前置判斷。
- **觀察指標**：當年 EPS vs 預估明年 EPS、Forward PE、EPS revision、需求 / 訂單 / backlog、CAPEX 用途、FCF 壓力、產能上線時間、新聞情緒、DCF intrinsic value、implied fair PE、便宜原因是衰退還是擴大投資。
- **相關文章**：宋分-20260207-forward-pe-market-decoding-c6a2a228；宋分-20260209-dcf-buyside-valuation-process-32a33181
- **可套用 ticker / 產業**：無具名 ticker；成長型科技股、AI / data center 擴產公司、財報後 EPS 下修但長期需求仍需驗證的 re-rating 標的。

## DCF 安全檢查 / PE 交易語言框架

- **一句話定義**：傳統 buyside 估值不是直接跑 DCF：先用產業、公司與財報推導 EPS / cash flow，再用 DCF 作 downside protection 與 intrinsic value 檢查，接著用 PEG 檢查市場是否誤解成長持續性，最後用 Forward PE / exit multiple 判斷 public market 何時重新定價。

- **觀察指標**：EPS model、cash flow、discount rate / WACC、intrinsic value、terminal value sensitivity、PEG、growth duration、implied fair PE、current Forward PE、target multiple、market expectation revision、fundamental thesis break、FCF / EV coverage、exit multiple。
- **相關文章**：宋分-20260209-dcf-buyside-valuation-process-32a33181；宋分-20260219-peg-market-decoding-growth-mispricing-ac62ac3f；宋分-20260224-dcf-pe-target-rerating-resale-market-d1cf3e16

- **來源**：`KOL/宋分/articles/20260206_amzn_capex_cycle3_valuation_thermometer.md`
- **一句話定義**：當市場因 hyperscaler CAPEX 暴增、FCF 轉弱或 EPS 被壓而恐慌時，先檢查 backlog / RPO / customer commitments 是否已領先 revenue 加速，再判斷這是過度投資還是下一輪 AWS / AI capacity 建置期。
- **適用情境**：大型雲端平台在重資本支出期被市場懲罰，但 backlog、RPO、客戶承諾或 cloud revenue 已出現領先改善；尤其適合 **AMZN** 這種市場把 CAPEX 當成 FCF 惡化來源，但 KOL 用歷史 cycle 判斷未來 AWS 營益率 / EPS 可能跳升的情境。
- **觀察指標**：CAPEX / revenue、PP&E additions、FCF margin、AWS revenue YoY、backlog / RPO YoY、customer commitments、AWS operating margin、EPS revision、depreciation、AI chips / Trainium / Bedrock demand、capacity utilization、management capex guide。
- **相關文章**：宋分-20260206-amzn-capex-cycle3-valuation-thermometer-a097e225
- **可套用 ticker / 產業**：**AMZN**；hyperscaler、cloud infrastructure、AI data center、AWS / cloud capacity cycle、CAPEX-heavy mega-cap growth。
