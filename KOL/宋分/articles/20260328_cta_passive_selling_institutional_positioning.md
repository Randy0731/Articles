## 主旨

宋分這篇把「股價一直跌，是不是有我不知道的壞消息」拆成更機構化的問題：這一波下跌到底是情緒、基本面，還是結構性賣壓？他提醒，大部分時候投資人看到的不是公司突然壞掉，而是 CTA、ETF / 被動資金、風控、再平衡、保險資金與公司回購黑窗期共同造成的被迫賣壓。

本文延續 2026-03-27「相對低點 / 安全區間」框架，但把它具體化成資金動作：機構不是一次買滿或一次賣光，而是把「訊號」對應到「部位」。壞消息不再破底時只做 10%-20% 測試單；強勢股撐住時拉到 30%-50% 核心部位；市場開始分化時才提高到 60%-80%；趨勢確認後才接近滿倉。

3/28 至 3/30 的補充 Note 則補上三個實務面：目前三個止跌訊號尚未同時成立、強勢股要在市場有壓力時找「跌不下去」而不是漲最多、資金派的「先看到資金再研究」只在主升段或資金剛擴散時有效，最後仍要回到財報、結構與競爭力驗證。

## 內部人賣股：先分辨制度性賣出與基本面變壞

宋分先拆散戶最容易恐慌的「內部人賣股」。他的結論是：內部人賣股不等於公司壞掉，很多時候是制度化、合法、計畫性的行為。

| 類型 | 解讀 |
|---|---|
| 10b5-1 計畫 | 內部人預先設定時間、數量、價格，計畫設好後不受當下新聞或股價影響 |
| 黑窗期 / 交易限制期 | 財報前後避免利用未公開資訊交易，交易窗口本來就受限 |
| 定期披露制度 | 內部人買賣需向主管機關報告，市場可看到誰在賣、賣多少、何時賣 |
| 公司政策 / 合約限制 | 有些公司要求分批賣、限制年度賣出比例，以避免一次性衝擊市場 |
| 個人資產配置 | 買房、繳稅、期權行使、多元化與風險降低都可能造成賣股 |

真正要警覺的是「連續大量主動加速賣出 + 基本面轉弱」。若只有內部人賣股，沒有需求、毛利率、訂單、現金流或 guidance 變差，不應直接寫成公司出問題。

## CTA 與其他結構性賣壓

宋分把 CTA 定義為用趨勢模型交易的資金。這類資金不是因為覺得公司不好，而是價格跌破趨勢線時系統自動賣。例如跌破 50 日均線減碼、跌破 100 日均線再賣、趨勢轉空則全出清。當很多模型使用類似趨勢指標，就會出現模型共振。

除了 CTA，他也列出幾種常被忽略的結構性賣壓：

| 賣壓來源 | 為什麼會賣 | 對盤面的影響 |
|---|---|---|
| ETF / 被動資金 | ETF 被贖回時必須賣出成分股 | 權重高、流動性好的好公司反而跌更兇 |
| 養老基金 / 主權基金 | 資產配置比例不能跑掉，需要再平衡 | 跟公司好壞無關，是機械式調整 |
| 主動基金風控 | 單股比例、回撤、波動限制觸發降部位 | 不是看壞，而是風控機制要求降槓桿 / 降持股 |
| 保險公司 / 長期資金 | 利率上升時債券相對吸引力提高 | 股票賣壓慢、穩、難察覺，形成上檔壓力 |
| 公司回購消失 | 財報不確定、blackout period、現金流壓力使回購暫停 | 買盤突然消失，投資人誤以為只有賣壓增加 |

整理者判斷：這段要和 2026-03-20 風險重新定價、2026-03-27 安全區間一起讀。股價下跌時要先問「誰被迫賣、為什麼一定要賣」，而不是只問是不是有未知壞消息。

## 賣壓鈍化的三個訊號

宋分沿用前一篇的三個止跌訊號：

| 訊號 | 含義 | 本篇補充 |
|---|---|---|
| 利空很多，但不再破底 | 該賣的可能賣得差不多 | 第一筆資金不是抄底，而是買一個觀察權 |
| 強勢股不再跌 | 錢沒有真的撤出市場 | 代表核心資金還願意留在原主線或優質資產 |
| 市場開始分化 | 從亂賣變成選股 | 基本面開始重新被定價，勝率提高 |

3/28 補充 Note `c-234532297` 將當時盤勢判為三個訊號尚未同時成立：指數跌破長期均線、仍創低或接近前低，代表 CTA / ETF / 風控賣壓可能還在進行；強勢股跌破長期均線代表核心資金也在撤；光通訊相對抗跌、儲存等板塊補跌，更像輪流補跌而不是健康分化。

## 訊號對應部位：10% -> 40% -> 70% -> 100%

宋分把機構動作拆成資金比例，重點是「風險下降才加碼」，不是「更便宜就加碼」。

| 階段 | 訊號 | 典型資金動作 | 用意 |
|---|---|---|---|
| 第一階段 | 不再破底 | 10%-20% 測試單 | 買一個觀察權，若再破底可快速降回 0%-5% |
| 第二階段 | 強勢股撐住 | 30%-50% 建立核心 | 風險從失控變成可觀察，開始建立核心部位 |
| 第三階段 | 市場開始分化 | 60%-80% 加碼 | 市場從亂殺轉向選股，基本面重新被定價 |
| 第四階段 | 趨勢確認 | 80%-100% 接近滿倉 | 指數上行、資金回流、趨勢成立後才放大到高部位 |

如果判斷錯，機構不是一次停損，而是把部位往回調。例如第一階段又破底，10% 可降回 0%-5%；第二階段訊號變差，40% 可降回 20%。這是風險控制，不是輸贏面子。

## 機構不會優先加碼跌最多的部位

本篇最重要的反散戶直覺，是機構加碼的不是跌最多的，而是最有機會先漲的。

| 類型 | 為什麼優先 |
|---|---|
| 最先不再跌的 Leader | 大盤還在震盪，但它已經不破低、橫盤或微漲，代表資金先回來 |
| 原本的強勢族群 / Core Holdings | AI、半導體、長期成長主軸若基本面和故事沒壞，機構會回補核心部位 |
| 新資金流入方向 / Rotation | 市場穩定後某些板塊開始漲，代表資金在卡新的主線 |

虧損部位不會因為虧損就優先加碼。機構先問邏輯還在不在；若邏輯還在、只是錯殺，才可能加碼，但不一定優先。若邏輯動搖、資金沒有回來，很多時候是不動，甚至降權重。

## 回檔前機構先賣什麼

3/29 補充 Note 說，若資金結構開始壞掉，機構不是隨便賣。通常先賣最容易賣的：大型股、流動性好、有很多人接的標的。因為第一優先不是報酬，而是控制風險。

接著會賣已經大漲、共識很高、擁擠交易，以及敘事已完成、未來沒有新故事、估值在高位的標的。小型股 / 流動性差的反而常是最後才動，因為一賣就會崩。

這也解釋為什麼有時候好公司先跌：不是因為它最差，而是因為它最容易被賣來降風險。

## 止跌順序不等於加碼順序

宋分補充，市場止跌常見順序是：

1. 權值股止跌：用於穩住整體風險，指數先穩。
2. 成長股開始動：原本主線、基本面沒壞、只是被情緒打下來，是主要報酬來源。
3. 高 beta 噴出：通常在趨勢確認後，用於放大績效。

散戶常在前面不敢買，最後去追最兇的高 beta；機構則用權值股止血、用成長股賺錢、用高 beta 放大績效。這裡仍是框架，不是要求投資人照比例或照順序操作。
