# Research

此目錄放「非 KOL、非單一個股」的研究資料。

## 使用情境

- 產業報告
- 宏觀研究
- 多公司橫向比較
- 跨 KOL 比較筆記（使用者明確要求入庫時）

## 資料夾

```text
Research/
├── raw/
├── notes/
├── industry/
├── macro/
└── comparisons/
```

## 原則

- 若資料由特定 KOL 發出，主整理仍放 `KOL/<KOL>/`
- 若資料以單一個股為主，主整理放 `Stocks/<Ticker>/`
- 若資料是產業、宏觀或橫向比較，放 `Research/`
