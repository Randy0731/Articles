# 筆記與索引模板

格式硬規格見 [docs/FORMAT_SPEC.md](docs/FORMAT_SPEC.md)。本文件只提供可複製的範例模板。

## 合併筆記檔頭模板

```md
# [KOL 或 TICKER] 筆記

此檔採 append-only：新整理文章加在檔尾；下方文章索引按發文時間倒序維護。

## 文章索引（按發文時間倒序）

| 發文時間 | 整理日期 | 標題 | source_id | 相關 ticker | 主題 tags | 備註 |
|---|---|---|---|---|---|---|

---

## 內文（append-only）
```

## 索引表模板

### ticker index

```md
| Ticker | 日期 | KOL | 來源 | 立場 | 訊號強度 | 價位 / 性質 | 催化劑 | 風險 |
|---|---|---|---|---|---|---|---|---|
```

### theme index

```md
| 主題 tag | 日期 | KOL / 來源 | 標題 | 核心觀點 | 相關 ticker | 框架 / 催化劑 | 筆記檔 |
|---|---|---|---|---|---|---|---|
```

### source index

```md
| source_id | 入庫日期 | KOL / 來源 | 標題 | 發文時間 | 類型 | 原始檔案 / URL | raw 路徑 | 筆記檔 | OCR 狀態 | 狀態 | duplicate_of / 備註 |
|---|---|---|---|---|---|---|---|---|---|---|---|
```

### catalyst index

```md
| 日期 | 事件 | 相關 ticker | KOL / 來源 | 出處 | 重要性 | 狀態 | 結果 / 後續 | 最後檢查 |
|---|---|---|---|---|---|---|---|---|
```

### watchlist index

```md
| Ticker | 首次日期 | 最近日期 | 來源 KOL | 觸發原因 | 目前狀態 | 升級條件 |
|---|---|---|---|---|---|---|
```
